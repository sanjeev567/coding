What is javaScript

javaScript is a clientside scripting language
It is used to make webpages alive
It is used to make programmitically perform action within the page
Initially it is called livescript
Java is very popular on that time so they decided to keep its name as younger brother of java

what can we do with javaScript

javaScript can execute not only in the browser but also in the server side javaScript can do several task instead of web language.

What can a javaScript do in-Browser.

Add new Html and change exiting html from dom(document module you can say page) eg- we can change our home, services, about and contact which is wrtten inside nav>ul>li>a; React to events (like respose from the server, pressing of any key, cursor movements etc. (like while clicking on button it will submit the form in the server)) Ajax requests(respond on send or receive info from the server or client without reloading your page) get and set cookies(cookies are some space or memory where the info about the user have in order to give better experience to the user.)

What can't do javaScript in-Browser.

it can't read/write anything in the local disk file(harddisk) it can't read or send xml request into users emails to read or write it it can't send xml request to the other site (like fb.com) to get content until that site do not allow it to do.
